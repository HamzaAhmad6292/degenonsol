# GIFs Folder

This folder should contain three GIF files for the otter chat page:

1. **happy.gif** - Displayed when token price increases
2. **sad.gif** - Displayed when token price decreases or stays the same
3. **idle.gif** - Displayed as the initial/default state

## File Naming
- All GIFs must be named exactly as shown above (lowercase)
- Supported formats: `.gif`

## Usage
The chat page will automatically load these GIFs based on token price movements:
- Price goes up → `happy.gif`
- Price goes down or stays same → `sad.gif`
- Initial load → `idle.gif`

## Recommended Specifications
- Format: GIF (animated)
- Recommended size: 400-600px width/height
- File size: Keep under 2MB for optimal performance
- Background: Transparent or matching the luxury theme
